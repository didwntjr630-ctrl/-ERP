# 삼양ERP 입출고 등록 템플릿

이 폴더에는 입출고 등록 화면 템플릿 파일이 있습니다.

파일 구조:
- `index.html` : 화면 구성과 요소를 담은 파일
- `styles.css` : 화면 모양(색, 여백, 테이블 스타일)을 정하는 파일
- `script.js` : 버튼 클릭과 입력 초기화 기능을 담당하는 파일

사용 방법:
1. `index.html`을 웹 브라우저로 열면 화면을 볼 수 있습니다.
2. 버튼을 눌러서 기본 동작을 확인할 수 있습니다.
3. 필요한 내용은 `index.html`, `styles.css`, `script.js`를 각각 수정하세요.

주의:
- 기존 `CLAUDE.md` 파일은 건드리지 않았습니다.
- 데이터가 삭제되지 않도록 새로운 파일만 생성했습니다.
